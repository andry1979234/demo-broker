# Demo Telegram Broker (Kasper Coin)

Простой демонстрационный проект: внутри Telegram можно открыть WebApp (или напрямую в браузере) — там отображается случайно генерируемый свечной график и можно открывать сделки Buy/Sell с экспирацией.

Запуск локально:

1. Установить зависимости

```bash
npm install
```

2. Запустить сервер

```bash
PORT=3000 node server.js
```

3. (Опционально) Запустить бота (для WebApp кнопку). Установите `TELEGRAM_BOT_TOKEN` и `WEBAPP_URL` (адрес, где доступен ваш WebApp, например через ngrok)

```bash
TELEGRAM_BOT_TOKEN=XXX WEBAPP_URL=https://abcd.ngrok.io node bot.js
```

Технические детали:

- Сервер: `server.js` — генерирует свечи каждую секунду и хранит торги в памяти.
- Фронтенд: `public/index.html`, `public/app.js` — использует Lightweight Charts для отображения свечей.
- Бот: `bot.js` — отправляет пользователю кнопку для открытия WebApp.

Ограничения:

- Хранение в памяти — только для демо. Для реального использования нужен БД и авторизация.
- Используется случайный генератор цен для тестов.
